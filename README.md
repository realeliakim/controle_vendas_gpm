## Controle de Vendas

## :computer: Passos para instalação

```bash

# Clone este repositório
$ git clone git@github.com:realeliakim/controle_vendas_gpm.git


#Adicionar dependências
$ composer intall
$ npm install && npm run dev


#Executar migrations e seeds
$ php artisan migrate
$ php artisan db:seed


#Executar servidor
$ php artisan serve

```
